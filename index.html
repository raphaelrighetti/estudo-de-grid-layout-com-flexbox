A ideia desse mini projeto foi fazer um projeto simples para testar como funciona criar um grid layout usando o display: flex;.


Nesse caso pude usar o justify-content: space-between;, pois o número de elementos por coluna está igual, mas estou ciente de que se precisar é melhor usar medidas mais precisas usando porcentagem nas margins e etc.
